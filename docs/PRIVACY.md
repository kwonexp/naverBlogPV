# 개인정보 처리방침 (초안)

본 문서는 Chrome 확장 프로그램 **Naver Blog Rank Toolkit**의 데이터 처리 방침을 설명합니다.

## 시행일

- 2026-02-28

## 수집하는 정보

- 본 확장 프로그램은 사용자의 개인식별정보를 수집하지 않습니다.
- 본 확장 프로그램은 별도의 사용자 계정, 비밀번호, 결제정보를 수집하지 않습니다.

## 처리하는 데이터

- 사용자가 네이버 블로그 통계 화면에서 `Extract`를 실행할 때,
- 화면에 표시된 순위 테이블 데이터(예: 날짜, 순위, 제목, 조회수, 타입, 작성일)를 브라우저 내부에서 처리합니다.

## 데이터 전송 및 공유

- 추출/집계 데이터는 외부 서버로 전송되지 않습니다.
- 제3자에게 판매/공유하지 않습니다.

## 권한 사용 목적

- `downloads`: CSV 파일 저장
- `webNavigation`: 현재 탭 프레임 탐색
- `clipboardWrite`: 결과를 클립보드로 복사
- `https://blog.stat.naver.com/*`: 대상 통계 화면에서 데이터 추출

## 데이터 보관

- 확장 프로그램은 자체 서버에 데이터를 저장하지 않습니다.
- 사용자가 저장한 CSV/클립보드 데이터는 사용자의 로컬 환경에서만 관리됩니다.
